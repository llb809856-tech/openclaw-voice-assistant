## 02 为什么是 QQ？

很多人可能会问：为什么腾讯选择 QQ 而不是微信？

**原因有三：**

### 🔹 QQ 的开放基因
- QQ 开放平台已有成熟的机器人生态

### 🔹 QQ 的用户群体
- 年轻用户占比高，对 AI 接受度强
- 学生群体庞大，学习/创作需求旺盛
- 社群活跃，易于传播

### 🔹 战略卡位
- 微信需要更谨慎的合规审查
- QQ 可以作为"试验田"快速迭代
- 抢占 AI Agent 入口，避免落后于竞争对手

## 05 接入教程：3 步搞定

**第一步：创建 QQ 机器人**
1. 访问 QQ 开放平台（q.qq.com）
2. 扫码登录 QQ 账号
3. 点击"创建机器人"
4. 保存 AppID 和 AppSecret

**第二步：部署 OpenClaw**
1. 选择部署方式（本地/云端）
2. 安装 QQ 插件：`openclaw plugins install @sliverp/qqbot@latest`
3. 配置 AppID 和 AppSecret

**第三步：测试上线**
1. 启动 OpenClaw 服务
2. 在 QQ 中@机器人测试
3. 确认功能正常后上线

## 06 风险与注意事项

虽然机会很大，但有几点需要注意：

### ⚠️ 安全风险
- 不要将 AppSecret 泄露给他人
- 定期更新密码和密钥
- 避免在公网暴露 OpenClaw 实例

### ⚠️ 合规风险
- 不要用于违法违规用途
- 遵守平台规则（QQ、微信等）
- 涉及敏感领域需谨慎（医疗、法律、金融）

### ⚠️ 技术风险
- OpenClaw 仍在快速迭代中
- 部分功能可能不稳定
- 建议先在测试环境验证
