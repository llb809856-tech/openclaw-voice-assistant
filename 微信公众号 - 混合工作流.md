##  各工具使用说明

### 1. wechat-mp-writer（写作 + 润色）

**位置**: `~/.agents/skills/wechat-mp-writer/SKILL.md`

**核心功能**:
- ✅ 热点选题建议
- ✅ 多种风格撰写（技术干货、故事叙事、观点评论）
- ✅ **AI 去味润色**（去除机器感，加入口语化表达）
- ✅ 配图建议

**AI 去味检查清单**:
- [ ] 没有"首先/其次/最后/综上所述"等模板化连接词
- [ ] 没有过于完美的三段式结构
- [ ] 有具体的个人经历或观察
- [ ] 有情感词（惊喜、崩溃、感动、无语）
- [ ] 有口语化表达（"说白了..."、"你可能想不到..."）
- [ ] 长短句交错，不是每段都一样长
- [ ] 有适当的括号补充（（笑）、（别问我怎么知道的））

---

### 2. bm.md（排版格式化）

**API**: `POST https://bm.md/api/markdown/render`

**参数**:
```json
{
  "markdown": "文章内容",
  "markdownStyle": "green-simple",
  "platform": "wechat",
  "enableFootnoteLinks": true,
  "openLinksInNewWindow": true
}
```

**输出**: 带内联 CSS 的 HTML，可直接复制到公众号后台

**主题颜色**:
- 绿色主题：`#2bae85`
- 黑红主题：`#c41e3a`（中国红）+ `#1a1a1a`（深黑）

---

### 3. wechat-publisher-yashu（发布）

**位置**: `~/.agents/skills/wechat-publisher-yashu/SKILL.md`

**配置**: `~/.agents/skills/wechat-publisher-yashu/config.json`

## 🎯 最佳实践

### 写作流程

1. **确定主题**：用 wechat-mp-writer 生成选题建议
2. **撰写初稿**：选择合适风格（技术/故事/观点）
3. **AI 去味**：执行润色，检查去味清单
4. **排版**：用 bm.md 转换为 HTML（黑红主题）
5. **发布**：用 wechat-publisher-yashu 发布到草稿箱
6. **预览**：公众号后台预览，手动调整样式

### 排版风格

**黑红主题特点**:
- 主标题：黑色 + 红色底线
- 小标题：红色 + 左边框
- 重点文字：红色加粗
- 引用块：浅红背景 + 红色左边框
- 分隔线：红色渐变
