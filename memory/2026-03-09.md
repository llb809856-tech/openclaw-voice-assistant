## 📕 小红书发布步骤

```bash
# 1. 启动 MCP 服务
cd /tmp/xiaohongshu-mcp && ./xiaohongshu-mcp-server &

# 2. 配置 mcporter
npx mcporter config add xiaohongshu-mcp http://localhost:18060/mcp

# 3. 检查登录
npx mcporter call xiaohongshu-mcp.check_login_status

# 4. 准备内容（真实换行，不用 \n）
cat > content.txt << 'EOF'
第一行
第二行
EOF

# 5. 发布
npx mcporter call xiaohongshu-mcp.publish_content \
  'title:"标题"' \
  'content:"第一行
第二行"' \
  'images:["/图片/绝对路径.jpg"]' \
  'tags:["标签 1","标签 2"]'
```
