## 💡 提示词优化技巧

### 提升真实感

```
添加：
- "shot on Hasselblad X2D"（相机型号）
- "85mm lens, f/5.6"（镜头参数）
- "professional product photography"（摄影类型）
- "ultra detailed leather texture"（细节描述）

避免：
- "illustration"、"drawing"、"art"
- "beautiful"、"amazing"（空洞形容词）
- 过多情感词
```

### 控制颜色

```
精确描述：
- "black goatskin leather"（黑色小山羊皮）
- "dark gray marble surface"（深灰色大理石）
- "morandi color palette"（莫兰迪色系）

避免：
- "colorful"、"vibrant"（太宽泛）
- 多种颜色冲突
```

### 修复手部问题

```
AI 常画不好手，添加：
- "perfect hands, 5 fingers"
- "natural hand pose"
- "no deformed fingers"

负面提示词加：
- "deformed hands, extra fingers, missing fingers"
```

## ⚠️ 常见问题解决

### Q1：手套形状不对

```
解决：
- 添加 "laid flat"（平放）或 "worn on hand"（佩戴）
- 指定 "left hand" 或 "right hand"
- 用 "product photography" 强调是产品图
```

### Q2：皮革质感像塑料

```
解决：
- 添加 "natural leather grain visible"
- 添加 "visible pores and texture"
- 添加 "matte finish, not shiny"
- 负面提示词加 "plastic look, shiny surface"
```

### Q3：光线太平

```
解决：
- 明确 "45 degree side lighting"
- 添加 "soft shadows showing texture"
- 添加 "rim light from back"
- 避免 "flat lighting, even lighting"
```

### Q4：背景太乱

```
解决：
- 添加 "clean neutral background"
- 添加 "minimalist composition"
- 添加 "40% negative space"
- 负面提示词加 "cluttered background, busy scene"
```

### Q5：场景像室内

```
解决：
- 添加 "outdoor", "street", "sidewalk"
- 添加 "natural daylight", "winter atmosphere"
- 负面提示词加 "indoor, restaurant interior, office interior"
```
