## 3. 功能清单

### v1.0 - 基础功能
| 功能 | 触发条件 | 动作 |
|------|----------|------|
| 早安模式 | 检测到"人出现" + 时间 6-9点 | 播报天气 + 新闻 + 今日提醒 |
| 晚安模式 | 检测到"人出现" + 时间 22-24点 | 晚安问候 + 家里情况检查 |
| 在家检测 | 持续检测到人 | 记录在家状态 |
| 离家警报 | 长时间无人 → 突然有人 | 发送警报 |

| 功能 | 说明 |

## 4. 视觉模型选择

| 模型 | 优点 | 缺点 | 成本 |
|------|------|------|------|
| **LLaVA (本地)** | 免费、私密、不需要网 | 理解力稍弱 | 0 |
| **GPT-4V** | 理解力强 | 花钱、需翻墙 | ¥0.5-2/次 |
| **Claude Vision** | 理解力强 | 花钱 | ¥0.5-2/次 |
| **阿里通义千问** | 便宜、国内可用 | 一般 | ¥0.1/次 |

**推荐**：先用本地 LLaVA 试试效果

## 5. 实施步骤

### 第一步：准备摄像头 (1天)
1. 下载 IP Webcam APP 到旧手机
2. 架设在房间合适位置
3. 获取视频流地址

### 第二步：搭建视觉识别 (1天)
1. 安装 LLaVA 或配置 API
2. 写定时截图脚本
3. 测试图像识别效果

### 第三步：编写触发逻辑 (1天)
1. 状态判断（人在不在、什么时候）
2. 对应动作（问候、警报）
3. 对接语音合成（TTS）

### 第四步：测试优化 (持续)
1. 调优识别准确率
2. 增加更多场景
3. 完善互动内容
