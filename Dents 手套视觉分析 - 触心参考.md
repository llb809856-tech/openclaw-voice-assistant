## 🎬 二、Dents 视觉拍摄手法分析

### 2.1 灯光配置 ⭐⭐⭐⭐⭐

**核心技法：45°侧光 + 柔光箱**

```
主光位置：
- 角度：45°侧光（从产品侧面 45 度角照射）
- 高度：略高于产品（向下 30°）
- 距离：约 1-1.5 米
- 附件：柔光箱或 shoot-through umbrella

辅光位置：
- 角度：对面侧 45°
- 强度：主光的 1/2 或 1/4
- 作用：减少阴影，保留细节

轮廓光（可选）：
- 位置：产品后方
- 作用：勾勒手套边缘，突出皮质纹理
```

**为什么用 45°侧光？**
- ✅ 突出皮革天然纹理
- ✅ 创造立体感和深度
- ✅ 避免平面光显得廉价
- ✅ 阴影自然，有高级感

### 2.2 拍摄角度 📐

| 角度 | 使用场景 | 视觉效果 | 触心适用性 |
|------|---------|---------|-----------|
| **45°俯视** | 产品主图 | 展示整体形状 + 部分细节 | ⭐⭐⭐⭐⭐ 推荐 |
| **平视** | 穿戴效果 | 模拟真实使用视角 | ⭐⭐⭐⭐ 推荐 |
| **特写（微距）** | 皮质纹理/缝线 | 强调工艺细节 | ⭐⭐⭐⭐⭐ 强烈推荐 |
| **90°侧光** | 戏剧性效果 | 强对比，突出质感 | ⭐⭐⭐ 选择性用 |
| **Flat Lay（俯拍）** | 场景搭配 | 手套 + 配饰组合 | ⭐⭐⭐⭐ 推荐 |

### 2.3 构图技巧

#### Dents 常用构图

```
1. 三分法构图
   - 手套放在画面 1/3 处
   - 留白 40% 增加高级感
   - 视线引导向产品

2. 对角线构图
   - 手套斜放，形成对角线
   - 增加动感和视觉张力
   - 避免死板居中

3. 框架构图
   - 用手套盒/道具做前景框架
   - 增加层次感
   - 引导视线聚焦产品

4. 负空间（留白）
   - 大面积纯色背景
   - 产品占画面 30-40%
   - 高级感核心技巧
```

### 2.4 背景选择

| 背景类型 | 色值参考 | 适用场景 |
|---------|---------|---------|
| **深灰纹理** | `#2A2A2A` | 商务款、黑色手套 |
| **米白大理石** | `#F5F1ED` | 女款、浅色手套 |
| **深棕木纹** | `#3D2817` | 复古款、棕色手套 |
| **纯白无缝纸** | `#FFFFFF` | 电商主图 |
| **浅灰渐变** | `#E8E8E8 → #D0D0D0` | 产品细节图 |

**避坑指南**：
- ❌ 纯黑背景（吸光，显得沉闷）
- ❌ 高饱和颜色（抢产品注意力）
- ❌ 复杂图案（显得廉价）
- ❌ 反光材质（除非刻意追求）

## 🎨 三、Dents 色彩策略

### 3.1 整体色调

**Dents 色调特征**：
- 低饱和度（莫兰迪色系）
- 暖色调为主（营造温暖感）
- 高对比度（产品与背景）

### 6.4 后期调色参数（参考）

```
Lightroom/Capture One 参数：

基础调整：
- 曝光：0 或 -0.3（避免过曝）
- 对比度：+10 到 +20（增加层次）
- 高光：-20 到 -30（保留细节）
- 阴影：+10 到 +20（提亮暗部）
- 白色：+5 到 +10
- 黑色：-5 到 -10

色彩调整：
- 色温：偏暖 +5 到 +10（营造温暖感）
- 色调：0 或微偏品红 +5
- 饱和度：-5 到 -10（降低饱和度=高级感）
- 自然饱和度：0 或 +5

HSL 调整（针对皮革色）：
- 橙色饱和度：-10（降低皮革黄感）
- 橙色明度：+5（提亮皮革）
- 红色饱和度：-5（减少偏红）

细节增强：
- 锐化：+30 到 +40（突出纹理）
- 减少杂色：+10 到 +20（干净画面）
```

## 💡 七、关键洞察与行动建议

### 7.1 核心洞察

1. **灯光是高级感的核心**
   - 45°侧光是皮革产品的黄金标准
   - 柔光箱必不可少（避免硬阴影）
   - 轮廓光可以显著提升质感

2. **留白=高级感**
   - 产品占画面 30-40%
   - 背景简洁统一
   - 避免画面太满

3. **细节特写建立信任**
   - 微距拍摄皮质纹理
   - 展示缝线工艺
   - 触屏功能特写

4. **叙事比展示更重要**
   - Dents 讲 248 年历史
   - 触心要讲专利科技 + 都市优雅
