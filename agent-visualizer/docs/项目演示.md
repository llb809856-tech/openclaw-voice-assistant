# 🎮 Agent Visualizer - 项目演示文档

**创建时间**: 2026-03-08  
**版本**: v0.1.0 MVP  
**状态**: ✅ 可演示

## 📋 产品定位

**游戏化 AI Agent 可视化管理平台**

让每个 AI Agent 拥有：
- 🎨 独特的 3D 形象
- 📊 实时的状态显示
- 💰 可视化的 Token 消耗
- 🏆 有趣的游戏化元素

---

## 🎯 核心功能演示

### 1️⃣ Agent 形象系统

每个 Agent 都有独特的形象和表情：

| Agent | 形象 | 状态 | 说明 |
|-------|------|------|------|
| 小林 | 🤖 | 思考中 | 运营总监 Agent |
| 爬虫 | 🕷️ | 工作中 | 数据采集 Agent |
| 分析 | 📊 | 空闲 | 数据分析 Agent |
| 写作 | ✍️ | 工作中 | 内容创作 Agent |
| 客服 | 💬 | 空闲 | 客服助手 Agent |

**状态动画：**
- 😴 空闲 - 呼吸灯效果
- 🤔 思考 - 摇头动画
- 💪 工作 - 快速震动
- 😵 错误 - 红色闪烁

---

### 2️⃣ Token 可视化

实时显示每个 Agent 的 Token 消耗：

```
┌─────────────────────┐
│  💰 已用：1,692     │
│  ❤️ 剩余：80%      │
└─────────────────────┘
```

**预警系统：**
- 🟢 >50% - 绿色安全
- 🟡 20-50% - 黄色警告
- 🔴 <20% - 红色危险 + 提示音

---

### 3️⃣ 实时监控面板

**统计卡片：**
- 💰 Token 总消耗
- ✅ 完成任务数
- 🟢 运行中 Agent 数
- ⚡ 整体效率

**活动日志：**
```
14:32 🤖 小林 完成了 AI 日报发送任务 ✅
14:30 🕷️ 爬虫 开始抓取竞品数据 ℹ️
14:28 📊 分析 生成了销售报告 ✅
```

---

### 4️⃣ 3D Avatar 系统（开发中）

使用 Three.js 渲染的 3D 机器人：

- **浮动动画** - 轻微上下浮动
- **状态动作** - 根据状态有不同动作
- **鼠标交互** - 头部跟随鼠标转动
- **自动旋转** - 360 度展示

## 🛠️ 技术演示

### 后端 API

**启动命令：**
```bash
cd backend
python3 main.py
```

**API 端点：**
```
GET  /api/agents       - 获取所有 Agent
GET  /api/agents/{id}  - 获取单个 Agent
GET  /api/logs         - 获取活动日志
GET  /api/stats        - 获取统计数据
GET  /api/achievements - 获取成就列表
WS   /ws               - WebSocket 实时推送
```

**测试示例：**
```bash
curl http://localhost:8000/api/stats
```

**返回数据：**
```json
{
  "tokenTotal": 1692,
  "tasksCompleted": 4,
  "runningAgents": 2,
  "totalAgents": 5,
  "efficiency": 98.5
}
```

---

### 前端演示

**启动命令：**
```bash
cd frontend
npm install
npm run dev
```

**演示页面：**
```bash
open demo.html
```

**技术栈：**
- React 18 + 组件化
- TailwindCSS 样式
- Framer Motion 动画
- Three.js 3D 渲染
- WebSocket 实时更新
