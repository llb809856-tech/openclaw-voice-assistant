## 3. 常见问题 FAQ 文档

### 一、购物相关

**Q1：如何下单购买？**
```
A：您可以通过以下方式下单：
1. 淘宝/抖音店铺：搜索【店铺名称】进入店铺选购
2. 微信小程序：搜索【小程序名称】
3. 官网：www.xxx.com

下单流程：选择商品 → 选择规格 → 加入购物车 → 结算 → 付款
```

**Q2：支持哪些支付方式？**
```
A：我们支持以下支付方式：
• 支付宝
• 微信支付
• 花呗分期（3/6/12 期）
• 信用卡
• 抖音月付

部分商品支持先用后付服务～
```

**Q3：可以修改订单信息吗？**
```
A：订单信息修改规则：
✅ 发货前：可修改收货地址、联系电话、备注信息
❌ 发货后：无法修改，需联系快递公司

修改方式：联系客服提供订单号和新信息，我们帮您处理
```

**Q4：下单后多久发货？**
```
A：发货时效说明：
• 现货商品：付款后 24-48 小时内发货
• 预售商品：按页面标注的预售时间发货
• 定制商品：7-15 个工作日

📍 发货仓库：[城市]
🚚 合作快递：中通/圆通/顺丰（可备注指定）
```

### 二、物流相关

**Q5：发什么快递？可以指定吗？**
```
A：默认合作快递：
• 普通订单：中通/圆通（包邮）
• 加急订单：顺丰（需补差价）
• 偏远地区：邮政 EMS

💡 下单时可备注指定快递，我们会尽量安排～
```

**Q6：多久能收到货？**
```
A：配送时效参考：
• 江浙沪：1-2 天
• 华北/华南：2-3 天
• 西南/西北：3-5 天
• 东北/内蒙：4-6 天
• 新疆/西藏：5-7 天

⚠️ 节假日、大促期间可能延迟 1-2 天
```

**Q7：物流一直不更新怎么办？**
```
A：物流异常处理：
1. 等待 24 小时：可能是快递未及时扫描
2. 超过 48 小时：联系客服查询
3. 超过 7 天无更新：申请补发或退款

📞 客服会帮您联系快递公司核实处理
```

### 三、售后相关

**Q8：支持退换货吗？条件是什么？**
```
A：退换货政策：
✅ 7 天无理由退换（不影响二次销售）
✅ 15 天质量问题免费退换
✅ 30 天保修服务

❌ 不支持退换的情况：
• 已洗涤/穿着/使用
• 吊牌/包装拆除
• 定制/贴身商品

💰 运费险已投保，退货无忧
```

**Q9：退换货流程是怎样的？**
```
A：退换货流程：

【退货流程】
1. 订单页面申请售后
2. 选择【退货退款】
3. 填写退货原因
4. 寄回商品（保留原包装）
5. 填写退货单号
6. 仓库验收后退款

【换货流程】
1. 订单页面申请售后
2. 选择【换货】
3. 选择换货规格
4. 寄回原商品
5. 收到新商品

⏱ 处理时效：1-3 个工作日
```

**Q10：退款多久到账？**
```
A：退款时效：
• 审核时间：1-24 小时
• 退款到账：审核通过后 1-3 个工作日
• 原路退回：退至付款账户

💳 不同支付方式到账时间：
• 支付宝/微信：即时到账
• 银行卡：1-3 个工作日
• 花呗：1-2 个账单周期
```

## 4. 尺码推荐计算器

### 服装类尺码推荐逻辑

```

第一步：询问商品类型
客服：请问您想咨询哪类商品的尺码呢？
A. 上装（T 恤/衬衫/外套）
B. 下装（裤子/裙子）
C. 连衣裙
D. 内衣/家居服

第二步：根据类型询问关键数据

【A. 上装】
客服：请告诉我您的：
1. 身高（cm）
2. 体重（kg）
3. 肩宽（可选，cm）
4. 胸围（可选，cm）
5. 平时穿什么码？（S/M/L/XL）
6. 喜欢宽松还是修身？

【B. 下装】
客服：请告诉我您的：
1. 身高（cm）
2. 体重（kg）
3. 腰围（cm）
4. 臀围（可选，cm）
5. 平时穿什么码？
6. 喜欢高腰还是中腰？

【C. 连衣裙】
客服：请告诉我您的：
1. 身高（cm）
2. 体重（kg）
3. 胸围（cm）
4. 腰围（cm）
5. 平时穿什么码？
6. 喜欢什么长度？（短款/中长款/长款）

【D. 内衣/家居服】
客服：请告诉我您的：
1. 身高（cm）
2. 体重（kg）
3. 上胸围/下胸围（内衣专用）

第三步：根据数据推荐尺码

根据您的偏好：
• 喜欢宽松：建议选大一码
```

### 售后特殊情况处理

```
【特殊情况 1】商品已使用/洗涤
处理流程：
1. 询问使用情况和使用时间
2. 要求提供照片/视频
3. 判断是否影响二次销售
4. 如影响：协商部分退款或拒绝退换
5. 如不影响：正常处理

【特殊情况 2】超过售后期限
处理流程：
1. 确认购买时间和申请时间
2. 超 7 天但在 15 天内：可换货不可退货
3. 超 15 天但在 30 天内：可维修
4. 超 30 天：建议付费维修或特殊申请

【特殊情况 3】客户情绪激动/投诉
处理流程：
1. 安抚客户情绪，表达歉意
2. 耐心倾听客户诉求
3. 记录问题详情
4. 升级主管处理（必要时）
5. 提供补偿方案（优惠券/小礼品）
6. 跟进至问题彻底解决

【特殊情况 4】恶意退换/差评威胁
处理流程：
1. 保留所有沟通记录
2. 要求提供完整证据
3. 按平台规则处理
4. 必要时申请平台介入
5. 上报风控部门备案

【特殊情况 5】大件/特殊商品
处理流程：
1. 确认商品类型和尺寸
2. 安排上门取件（如支持）
3. 确认包装要求
4. 购买运输保险
5. 全程跟踪物流
```
